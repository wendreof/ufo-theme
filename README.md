# 🌌 UFO Theme

**UFO Theme** is a collection of visually stunning and cosmic-inspired themes for Visual Studio Code. Whether you're a developer who loves the mysteries of the universe or just someone looking for a unique and immersive coding experience, UFO Theme is here to take your editor to another galaxy! 🚀

---

## ✨ Themes Included

### 1. 🌠 **Varginha** (Default)
Inspired by the mysterious extraterrestrial colors, this dark theme features sleek shades of gray, vibrant red highlights, and complementary tones for a modern, otherworldly vibe.

### 2. 👽 **Alien Green**
A bold and vibrant dark theme inspired by alien aesthetics, featuring striking shades of green, black, and yellow for a unique and eye-catching look.

### 3. 🌌 **Dark Blue**
A calming dark theme with deep blues, soft cyan, and complementary colors, perfect for a professional yet serene coding environment.

### 4. 🌑 **Dark**
A classic dark theme with deep blacks, light grays, and vibrant accents like orange and lime green for a sleek and modern feel.

### 5. ☀️ **Light**
A clean and modern light theme with bright whites, dark grays, and complementary colors for a fresh and clear interface.

---

## 🚀 How to Use

1. **Clone or Download** this repository.
2. Open the folder in **Visual Studio Code**.
3. Go to the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of the window.
4. Click on the "..." menu in the Extensions view and select **"Install from VSIX..."**.
5. Select the `.vsix` file from the repository to install the theme.
6. Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P` on macOS) and type `Preferences: Color Theme`.
7. Select your desired **UFO Theme** from the list and enjoy the cosmic vibes! 🌌
